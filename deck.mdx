## Configurando Ambiente

Global:

```
git config --global user.name "Larissa"
git config --global user.email "contato@prov.com"
```

Para visualizar suas configurações:

```
git config user.name
Larissa
```

Para visualizar TODAS as informações:

```
git config --list
```

Editor do git (opcional):

```
git config --global core.editor code
```

### SSH keys

- RSA, DSA, ECDSA e ED25519;

```
ssh-keygen -t rsa -b 4096 -C "email@hue.br"
```

-t [type]

-b [bits]

-C [comentário]

---

#### Copiando sua key:

-Mac OS:

```
pbcopy < ~/.ssh/id_rsa.pub
```

-WSL / GNU / Linux (requer o pacote xclip):

```
xclip -sel clip < ~/.ssh/id_ed25519.pub
```

-Git Bash no Windows:

```
cat ~/.ssh/id_ed25519.pub | clip
```

## Inicializando um repositório

```
mkdir git-guide
cd git-guide/
git init

Initialized empty Git repository in ../../../git-guide/.git

cd .git/
ls
HEAD   branches    config     description   hooks    info  objects  refs
```

HEAD -> Origem padrão atual

branches -> Branches existentes no projeto

config -> configuração do repositorio

description -> Descrição do projeto

hooks -> Gatilhos para fazer certas ações

## Desfazendo coisas

Antes do Stage:

```
git checkout nomedoarquivo
```

Depois do Stage:

```
git reset HEAD nomedoarquivo
```

Depois do commit:

```
git reset --soft --mixed --hard
```

soft: Modificações irão voltar para staged e pronto para commitar novamente

mixed: Modificações irão voltar para o modified

hard: Ignora a existencia do commit. Destroí todo o commit.

---

```
git reset --soft f0dada5e0c4f6c50bf2a4a57f6b39677827e5f06

git reset --mixed f0dada5e0c4f6c50bf2a4a57f6b39677827e5f06

git reset --hard f0dada5e0c4f6c50bf2a4a57f6b39677827e5f06
```
